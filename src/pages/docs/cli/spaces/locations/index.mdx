Use the `ably spaces locations` command group to manage [member locations](/docs/products/spaces/features/member-location) in a space.

## Subcommands

| Command | Description |
| ------- | ----------- |
| [`ably spaces locations set`](/docs/cli/spaces/locations/set) | Set your location in a space. |
| [`ably spaces locations subscribe`](/docs/cli/spaces/locations/subscribe) | Subscribe to location updates in a space. |
| [`ably spaces locations get-all`](/docs/cli/spaces/locations/get-all) | Get all current locations in a space. |
