Use the `ably channels update` command to update a message on an Ably Pub/Sub channel.

## Synopsis

<Code>
```shell
ably channels update <channel> <serial> <message> [flags]
```
</Code>

## Arguments

### `channel` <a id="channel"/> `(Required)`

The name of the channel containing the message.

### `serial` <a id="serial"/> `(Required)`

The serial of the message to update.

### `message` <a id="message"/> `(Required)`

The new message content. This can be plain text or JSON.

## Flags

### `--description` <a id="description"/>

A description for the update.

### `--encoding | -e` <a id="encoding"/>

The encoding of the message, such as `json/utf-8`.

### `--name | -n` <a id="name"/>

The event name for the updated message.

### `--client-id` <a id="client-id"/>

A client ID to use when updating the message.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Update a message:

<Code>
```shell
ably channels update my-channel "01a2b3c4d5@1234567890000-0" "Updated message content"
```
</Code>

Update a message with a description:

<Code>
```shell
ably channels update my-channel "01a2b3c4d5@1234567890000-0" '{"key": "new-value"}' --description "Fixed typo"
```
</Code>
