<Code>
```shell
ably push batch-publish [payload] [flags]
```
</Code>

Use the `ably push batch-publish` command to publish push notifications to multiple recipients in a batch.

## Arguments

Pass the following arguments to the `ably push batch-publish` command:

### `[payload]` <a id="payload"/>

A JSON array of push notification payloads. This can be provided as an inline JSON string, a file path, or `-` to read from stdin.

## Flags

You can optionally pass the following flags to the `ably push batch-publish` command:

### `--force | -f` <a id="force"/>

Skip confirmation when publishing to channels. Required when using `--json`. The default is `false`.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Publish with an inline JSON payload:

<Code>
```shell
ably push batch-publish '[{"recipient":{"deviceId":"device1"},"notification":{"title":"Hello"}},{"recipient":{"clientId":"user1"},"notification":{"title":"Hi"}}]'
```
</Code>

Publish from a file:

<Code>
```shell
ably push batch-publish @notifications.json
```
</Code>

Publish from stdin:

<Code>
```shell
cat notifications.json | ably push batch-publish -
```
</Code>
