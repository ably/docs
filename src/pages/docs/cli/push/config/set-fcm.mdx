Use the `ably push config set-fcm` command to configure FCM push notifications for an app.

## Flags

You can optionally pass the following flags to the `ably push config set-fcm` command:

### `--app` <a id="app"/>

The app ID to configure.

### `--service-account` <a id="service-account"/>

The path to the FCM service account JSON file. Required.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Configure FCM with a service account:

<Code>
```shell
ably push config set-fcm --service-account /path/to/service-account.json
```
</Code>
