Use the `ably spaces cursors get` command to get all current cursors in a space.

## Synopsis

<Code>
```shell
ably spaces cursors get <name>
```
</Code>

## Arguments

### `name` <a id="name"/>

The name of the space to get cursors for.

## Flags

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Get all cursors in a space:

<Code>
```shell
ably spaces cursors get my-space
```
</Code>

Get cursors in JSON format:

<Code>
```shell
ably spaces cursors get my-space --json
```
</Code>
