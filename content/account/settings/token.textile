Generate, manage, and revoke access tokens, ensuring secure access control by creating.

h3. Create a new access token

* Click on *Create New Access Token*.
* Fill out the required information:
** *Name:* _Assign a descriptive name._
** *Account:* _Assign an account._
** *Capabilities:* _Select the capabilities the token should have._
* *Create* the token.
* Copy the newly generated token and store it securely.

<aside data-type='note'>
<p>For security reasons, you will not be able to view the full token again after this point.</p>
</aside>

h3. Revoke a token

# Locate the token you wish to revoke.
# Click *revoke token* under the *settings* section.
# Confirm the action when prompted.


<aside data-type='note'>
<p>Revoking a token is irreversible. The token will no longer be valid for API access.</p>
</aside>
