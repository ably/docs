<Code>
```shell
ably rooms reactions subscribe <room> [flags]
```
</Code>

Use the `ably rooms reactions subscribe` command to subscribe to reactions in a chat room.

## Arguments

Pass the following arguments to the `ably rooms reactions subscribe` command:

### `<room>` <a id="room"/>

The name of the chat room to subscribe to reactions in.

## Flags

You can optionally pass the following flags to the `ably rooms reactions subscribe` command:

### `--duration | -D` <a id="duration"/>

The duration in seconds to subscribe for before automatically unsubscribing.

### `--client-id` <a id="client-id"/>

A client ID to use for the subscription.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Subscribe to reactions in a room:

<Code>
```shell
ably rooms reactions subscribe my-room
```
</Code>

Subscribe for a specific duration:

<Code>
```shell
ably rooms reactions subscribe my-room --duration 60
```
</Code>
