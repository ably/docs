Use the `ably spaces locks` command group to manage component locking in Ably Spaces.

## Subcommands

| Command | Description |
| ------- | ----------- |
| [`ably spaces locks acquire`](/docs/cli/spaces/locks/acquire) | Acquire a lock in a space. |
| [`ably spaces locks get`](/docs/cli/spaces/locks/get) | Get a lock or all locks in a space. |
| [`ably spaces locks subscribe`](/docs/cli/spaces/locks/subscribe) | Subscribe to lock events in a space. |
