Use the `ably config show` command to display the contents of the Ably CLI config file.

## Synopsis

<Code>
```shell
ably config show
```
</Code>

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Display the config file contents:

<Code>
```shell
ably config show
```
</Code>

Display the config file contents in JSON format:

<Code>
```shell
ably config show --json
```
</Code>
