<Code>
```shell
ably queues create --name <name> [flags]
```
</Code>

Use the `ably queues create` command to create a new queue for your Ably application.

## Flags

You can pass the following flags to the `ably queues create` command:

### `--name` <a id="name"/>

The name of the queue to create. This flag is required.

### `--app` <a id="app"/>

The app ID to create the queue for. If not specified, the currently selected app is used.

### `--max-length` <a id="max-length"/>

The maximum number of messages the queue can hold. The default value is `10000`.

### `--region` <a id="region"/>

The region to create the queue in. The default value is `us-east-1-a`.

### `--ttl` <a id="ttl"/>

The time-to-live for messages in the queue, in seconds. The default value is `60`. The maximum value is `3600`.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Create a queue:

<Code>
```shell
ably queues create --name "my-queue"
```
</Code>

Create a queue with custom settings:

<Code>
```shell
ably queues create --name "my-queue" --max-length 5000 --ttl 300
```
</Code>

Create a queue in a specific region:

<Code>
```shell
ably queues create --name "my-queue" --region eu-west-1-a
```
</Code>
