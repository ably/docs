Use the `ably status` command to check the current status of Ably services.

## Synopsis

<Code>
```shell
ably status [flags]
```
</Code>

## Flags

### `--open | -o` <a id="open"/>

Open the Ably status page in a browser.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Check the status of Ably services:

<Code>
```shell
ably status
```
</Code>

Check status with JSON output:

<Code>
```shell
ably status --json
```
</Code>
