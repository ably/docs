<Code>
```shell
ably apps list [flags]
```
</Code>

Use the `ably apps list` command to list all applications in the current account.

## Flags

You can optionally pass the following flags to the `ably apps list` command:

### `--limit` <a id="limit"/>

The maximum number of results to return. Defaults to `100`.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

List all apps in the current account:

<Code>
```shell
ably apps list
```
</Code>

List apps in JSON format:

<Code>
```shell
ably apps list --json
```
</Code>
