Use the `ably spaces locations` command group to manage locations in Ably Spaces.

## Subcommands

| Command | Description |
| ------- | ----------- |
| [`ably spaces locations get`](/docs/cli/spaces/locations/get) | Get all current locations in a space. |
| [`ably spaces locations set`](/docs/cli/spaces/locations/set) | Set location in a space. |
| [`ably spaces locations subscribe`](/docs/cli/spaces/locations/subscribe) | Subscribe to location updates for members in a space. |
