Use the `ably auth revoke-token` command to revoke an Ably token.

## Synopsis

<Code>
```shell
ably auth revoke-token <token> [flags]
```
</Code>

## Arguments

Pass the following arguments to the `ably auth revoke-token` command:

### `token` <a id="token"/>

The token to revoke.

## Flags

You can optionally pass the following flags to the `ably auth revoke-token` command:

### `--app` <a id="app"/>

The app ID the token belongs to. Uses the currently selected app if not specified.

### `--client-id | -c` <a id="client-id"/>

The client ID to revoke tokens for.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Revoke a specific token:

<Code>
```shell
ably auth revoke-token "xVLyHw.DRE6hQ..."
```
</Code>

Revoke tokens for a specific client ID:

<Code>
```shell
ably auth revoke-token "xVLyHw.DRE6hQ..." --client-id "user-123"
```
</Code>
