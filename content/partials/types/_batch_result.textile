A @BatchResult@ contains information about the results of a batch operation.

h4.
  default: Properties

- <span lang="default">successCount</span> := The number of successful operations in the request<br>__Type: @Number@__

- <span lang="default">failureCount</span> := The number of unsuccessful operations in the request<br>__Type: @Number@__

- <span lang="default">messages</span> := An array of results for the batch operation (for example, an array of "@BatchPublishSuccessResult@":/docs/api/realtime-sdk/types#batch-publish-success-result or "@BatchPublishFailureResult@":/docs/api/realtime-sdk/types#batch-publish-failure-result for a channel batch publish request)<br>__Type: Object[]__
