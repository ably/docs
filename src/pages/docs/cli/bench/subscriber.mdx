<Code>
```shell
ably bench subscriber <channel> [flags]
```
</Code>

Use the `ably bench subscriber` command to run a subscriber benchmark test on an Ably channel.

## Arguments

Pass the following arguments to the `ably bench subscriber` command:

### `<channel>` <a id="channel"/>

The name of the channel to subscribe to. This argument is required.

## Flags

You can optionally pass the following flags to the `ably bench subscriber` command:

### `--duration | -D` <a id="duration"/>

The duration of the subscriber benchmark in seconds.

### `--client-id` <a id="client-id"/>

The client ID to use for the benchmark.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Run a basic subscriber benchmark:

<Code>
```shell
ably bench subscriber my-channel
```
</Code>
