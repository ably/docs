```shell
ably push devices get <device-id>
```
</Code>

Use the `ably push devices get` command to get details of a push device registration.

## Arguments

Pass the following arguments to the `ably push devices get` command:

### `<device-id>` <a id="device-id"/>

The ID of the device to retrieve details for.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Get details of a device registration:

<Code>
```shell
ably push devices get "device123"
```
</Code>
