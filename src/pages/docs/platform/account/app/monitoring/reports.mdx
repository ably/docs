The Reports section in your Ably dashboard provides comprehensive analytics and insights for your application, enabling you to monitor usage patterns, analyze costs, and optimize your implementation. View your application reports in the [app dashboard](https://ably.com/accounts/any/apps/any/reports) under the Reports section. This interface offers both high-level summaries and detailed breakdowns of your application's activity.

## App Reports overview <a id="app-reports-overview"/>

The main reports interface displays analytics and insights organized into key areas that help you understand your application's performance and resource consumption.

### Available report sections <a id="available-sections"/>

The reports dashboard includes these main areas:

| Report section | Description | Status |
|---|---|---|
| App-specific insights | Detailed analytics for channel usage, connection patterns, and message flow analysis | Coming soon |
| Billable consumption | Comprehensive usage metrics and cost analysis | 1 report available |

### App-specific insights <a id="app-insights"/>

This upcoming feature will provide detailed analytics specifically for your application, including:

- Channel usage patterns and performance metrics
- Connection behavior analysis and geographic distribution  
- Message flow analysis and delivery patterns
- Performance optimization recommendations

### Billable consumption reports <a id="billable-consumption"/>

The billable consumption section provides detailed usage metrics and cost analysis through the Global usage report.

#### Global usage report <a id="global-usage"/>

The Global usage report offers a consolidated view of your application's realtime activity and efficiency, showing:

- Growth trends and traffic patterns
- Resource consumption breakdowns
- Billable usage patterns and optimization opportunities

Click on the Global usage report to access detailed consumption analytics and usage breakdowns.

## Global usage detailed view <a id="global-usage-detailed"/>

When you access the Global usage report, you see comprehensive analytics across multiple categories that help you understand your application's resource consumption and identify optimization opportunities.

### Billable messages <a id="billable-messages"/>

The billable messages section tracks all message traffic across different delivery methods:

| Message type | Description | Usage tracking |
|---|---|---|
| Inbound (Realtime) | Messages received via realtime connections | Real-time message volume and patterns |
| Inbound (REST) | Messages received via REST API | API-based message delivery metrics |
| Outbound (Realtime) | Messages delivered via realtime connections | Real-time message distribution |
| Outbound (REST) | Messages delivered via REST API | API response message volume |
| Persisted (History) | Messages stored for history retrieval | Message persistence and storage usage |
| Integrations | Messages processed through integrations | Integration-triggered message flow |
| Push notifications | Push notification delivery volume | Mobile and web push notification metrics |
| Non-billable (Batch & Conflation savings) | Messages optimized through Ably's efficiency features | Cost savings from optimization |

The total messages chart displays message volume trends over time, helping you identify usage patterns and peak activity periods.

### Billable connections <a id="billable-connections"/>

Connection analytics track your application's client connectivity patterns:

| Connection metric | Description | Purpose |
|---|---|---|
| Connection minutes | Total time clients remain connected | Measure connection duration and efficiency |
| Peak concurrent connections | Maximum simultaneous connections | Monitor connection scaling requirements |

The connection charts help you understand connection patterns and plan for scaling requirements.

### Billable channels <a id="billable-channels"/>

Channel usage metrics provide insights into your messaging infrastructure utilization:

| Channel metric | Description | Purpose |
|---|---|---|
| Channel minutes | Total time channels remain active | Track channel lifecycle and efficiency |
| Peak concurrent channels | Maximum simultaneous active channels | Monitor channel scaling patterns |

### Message type traffic <a id="message-type-traffic"/>

This section breaks down message activity by content type:

| Traffic type | Description | Analysis focus |
|---|---|---|
| Messages | Standard messaging traffic | Core messaging patterns |
| Presence | Presence event activity | User status and availability tracking |
| Objects | Object state synchronization | Data synchronization patterns |
| Annotations | Message annotation activity | Content enhancement usage |

### Cost optimization insights <a id="cost-optimization"/>

The reports include analysis of Ably's built-in optimization features that reduce your costs:

#### Delta bandwidth savings <a id="delta-savings"/>

| Optimization metric | Description | Benefit |
|---|---|---|
| Bandwidth savings from deltas | Data transfer reduction through delta compression | Network efficiency and cost reduction |
| Bandwidth savings percentage | Percentage of bandwidth saved through optimization | Efficiency measurement |

#### Batching and conflation savings <a id="batching-savings"/>

| Optimization feature | Description | Impact |
|---|---|---|
| Messages reduced | Number of messages eliminated through batching and conflation | Message volume optimization |
| Messages reduced percentage | Percentage of message reduction achieved | Efficiency and cost optimization measurement |

### Usage breakdown <a id="usage-breakdown"/>

The detailed usage breakdown provides granular analysis with comprehensive data tables showing:

| Metric | Description |
|---|---|
| Period | Time period for the data row with hourly timestamps and timezone |
| Total messages | Combined message volume across all types |
| Total bandwidth | Data transfer volume in MiB (Mebibytes) |
| Connection minutes | Total connection time |
| Peak concurrent connections | Maximum simultaneous connections |
| Channel minutes | Total channel active time |
| Peak concurrent channels | Maximum simultaneous channels |
| Inbound messages | Messages received by your application |
| Outbound messages | Messages sent by your application |
| Integrations | Messages processed through integrations |
| Push notifications | Push notification delivery count |

This tabular data (showing 744 rows in the example) enables detailed analysis of usage patterns and helps identify opportunities for further optimization.

#### Export functionality <a id="export-functionality"/>

The usage breakdown data can be exported as a CSV file for further analysis in external tools. CSV exports are automatically named with timestamps (for example, `app_stats_details_v2_0_1_2025-12-29T19_09_45.352702118Z.csv`) to help you organize and track different export versions.
