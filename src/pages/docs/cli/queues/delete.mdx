Use the `ably queues delete` command to delete a queue.

## Synopsis

<Code>
```shell
ably queues delete <queue-id> [options]
```
</Code>

## Arguments

### `queue-id` <a id="queue-id"/> <RequiredBadge />

The ID of the queue to delete.

## Options

### `--app` <a id="app"/>

The app ID that the queue belongs to. If not specified, the currently selected app is used.

### `--force | -f` <a id="force"/>

Skip the confirmation prompt and delete the queue immediately. Required when using `--json`.

### `--json` <a id="json"/>

Output results as compact JSON. Mutually exclusive with `--pretty-json`.

### `--pretty-json` <a id="pretty-json"/>

Output results in formatted JSON. Mutually exclusive with `--json`.

### `--verbose | -v` <a id="verbose"/>

Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Delete a queue:

<Code>
```shell
ably queues delete appAbc:us-east-1-a:foo
```
</Code>

Delete a queue without a confirmation prompt:

<Code>
```shell
ably queues delete appAbc:us-east-1-a:foo --force
```
</Code>

Delete a queue for a specific app:

<Code>
```shell
ably queues delete appAbc:us-east-1-a:foo --app "My App"
```
</Code>

Delete a queue and output the result in JSON format:

<Code>
```shell
ably queues delete appAbc:us-east-1-a:foo --json
```
</Code>
