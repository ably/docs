An @DeviceDetails@ is a type encapsulating attributes of a device registered for push notifications.

  java:    Members
  ruby:    Attributes

- <span lang="default">id</span> := unique identifier for the device generated by the device itself<br>__Type: @Strinf@__

- <span lang="default">clientId</span><span lang="ruby">client_id</span> := optional trusted "client identifier":/core-features/authentication#identified-clients for the device<br>__Type: @String@__

- <span lang="default">formFactor</span><span lang="ruby">form_factor</span> := form factor of the push device. Must be one of @phone@, @tablet@, @desktop@, @tv@, @watch@, @car@ or @embedded@<br>__Type: @String@__

- <span lang="default">metadata</span> := optional metadata object for this device. - <span lang="default">platform</span> := platform of the push device. Must be one of @ios@, @android@ or @browser@<br>__Type: @String@__

- <span lang="default">updateToken</span><span lang="ruby">update_token</span> := a token that can be used by typically a mobile device to "authenticate":#token-authentication with Ably later and "update an existing device registration":#update-device-registration (for example, for updating a GCM or FCM registration token)<br>__Type: @String@__
