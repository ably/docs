Use the `ably interactive` command to launch an interactive shell for the Ably CLI. This provides a REPL-like experience where you can run Ably CLI commands without prefixing them with `ably`.

## Synopsis

<Code>
```shell
ably interactive [options]
```
</Code>

<Aside data-type='note'>
This feature is in ALPHA and may change in future releases.
</Aside>

## Options

### `--json` <a id="json"/>

Output results as compact JSON. Mutually exclusive with `--pretty-json`.

### `--pretty-json` <a id="pretty-json"/>

Output results in formatted JSON. Mutually exclusive with `--json`.

### `--verbose | -v` <a id="verbose"/>

Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Launch the interactive shell:

<Code>
```shell
ably interactive
```
</Code>
