</Code>

Use the `ably spaces cursors get` command to get all current cursors in a space.

## Arguments

Pass the following arguments to the `ably spaces cursors get` command:

### `<name>` <a id="name"/>

The name of the space to get cursors for.

## Flags

You can optionally pass the following flags to the `ably spaces cursors get` command:

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Get all cursors in a space:

<Code>
```shell
ably spaces cursors get my-space
```
</Code>

Get cursors in JSON format:

<Code>
```shell
ably spaces cursors get my-space --json
```
</Code>
