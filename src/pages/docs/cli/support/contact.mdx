<Code>
```shell
ably support contact [flags]
```
</Code>

Use the `ably support contact` command to access Ably's contact information and support channels. This command opens your default web browser to the Ably contact page where you can find various ways to get in touch with the Ably support team.

## Flags

You can optionally pass the following flags to the `ably support contact` command:

### `--help | -h` <a id="help"/>

Show CLI help for this command.

## Output

The `ably support contact` command:

| Action | Description |
| ------ | ----------- |
| **Browser Opening** | Opens the Ably contact page in your default browser |
| **Contact Information** | Provides access to multiple support channels |

## Examples

Open the Ably contact page:

<Code>
```shell
ably support contact
```
</Code>

**Output:**
<Code>
```text
Opening https://ably.com/contact in your browser...
```
</Code>
