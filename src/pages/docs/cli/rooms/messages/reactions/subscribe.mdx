<Code>
```shell
ably rooms messages reactions subscribe <room> [flags]
```
</Code>

Use the `ably rooms messages reactions subscribe` command to subscribe to message reactions in a chat room.

## Arguments

Pass the following arguments to the `ably rooms messages reactions subscribe` command:

### `<room>` <a id="room"/>

The name of the chat room to subscribe to message reactions in.

## Flags

You can optionally pass the following flags to the `ably rooms messages reactions subscribe` command:

### `--raw` <a id="raw"/>

Get raw events instead of summaries. Defaults to `false`.

### `--duration | -D` <a id="duration"/>

The duration in seconds to subscribe for before automatically unsubscribing.

### `--client-id` <a id="client-id"/>

A client ID to use for the subscription.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Subscribe to message reactions in a room:

<Code>
```shell
ably rooms messages reactions subscribe my-room
```
</Code>

Subscribe and get raw events:

<Code>
```shell
ably rooms messages reactions subscribe my-room --raw
```
</Code>
