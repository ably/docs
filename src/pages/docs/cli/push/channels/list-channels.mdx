Use the `ably push channels list-channels` command to list channels with push subscriptions.

## Flags

You can optionally pass the following flags to the `ably push channels list-channels` command:

### `--limit` <a id="limit"/>

The maximum number of channels to retrieve. Defaults to `100`.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

List channels with push subscriptions:

<Code>
```shell
ably push channels list-channels
```
</Code>
