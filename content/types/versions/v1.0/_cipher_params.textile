A @CipherParams@ contains configuration options for a channel cipher, including algorithm, mode, key length and key. Ably client libraries currently support AES with CBC, PKCS#7 with a default key length of 256 bits. All implementations also support AES128.

Individual client libraries may support either instancing a @CipherParams@ directly, using <span lang="default">"@Crypto.getDefaultParams()@":/realtime/encryption#get-default-params</span><span lang="csharp">"@Crypto.GetDefaultParams()@":/realtime/encryption#get-default-params</span><span lang="ruby">"@Crypto.get_default_params()@":/realtime/encryption#get-default-params</span>, or generating one automatically when initializing a channel, as in "this example":/realtime/encryption#getting-started.
