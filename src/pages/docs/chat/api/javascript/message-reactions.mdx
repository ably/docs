The `MessageReactions` interface is used to add, delete, and subscribe to message reactions.

Access it via `message.reactions`.

---

## Add a reaction to a message <a id="add"/>

`message.reactions.add()`

Add a reaction to a message.

`add(params: AddMessageReactionParams): Promise<void>`

### Parameters

Use the following parameters:

| Parameter | Description | Type |
| --------- | ----------- | ---- |
| `params` | An object containing the reaction type. | [`AddMessageReactionParams`](#addmessagereactionparams) |

#### AddMessageReactionParams

The following `AddMessageReactionParams` can be set:

| Properties | Description | Type |
| ---------- | ----------- | ---- |
| `type` | The type of reaction to add. | String |

### Returns

Returns a promise that resolves when the reaction has been added. On failure, the promise is rejected with an [`ErrorInfo`](/docs/chat/api/javascript/error-info).

<Code>
```javascript
await message.reactions.add({ type: '👍' });
```
</Code>

## Subscribe to message reaction events <a id="subscribe"/>

`message.reactions.subscribe()`

Subscribe to reaction events for this message.

`subscribe(listener: MessageReactionListener): Subscription`

### Parameters

Use the following parameters:

| Parameter | Description | Type |
| --------- | ----------- | ---- |
| `listener` | Callback that will be called when a message reaction event is received. | [`MessageReactionListener`](#messagereactionlistener) |

#### MessageReactionListener

A listener that listens for message reaction events.

`MessageReactionListener: (event: MessageReactionSummaryEvent) => void`

It uses the following parameters:

| Parameter | Description | Type |
| --------- | ----------- | ---- |
| `event` | A message reaction summary event. | [`MessageReactionSummaryEvent`](#messagereactionsummaryevent) |

#### MessageReactionSummaryEvent

An interface that represents a summary of message reactions. This event aggregates different types of reactions for a specific message.

It has the following properties:

| Property | Description | Type |
| -------- | ----------- | ---- |
| `type` | The type of the reaction event. | [`MessageReactionEventType`](#messagereactioneventtype) |
| `messageId` | The ID of the message that the reactions are for. | String |
| `reactions` | A map of reaction types to their counts and metadata. | [`MessageReactions`](#messagereactions) |

#### MessageReactionEventType

An enum that represents the types of message reaction events.

It has the following members:

| Member | Description |
| ------ | ----------- |
| `ReactionAdded` | A reaction was added to the message. |
| `ReactionRemoved` | A reaction was removed from the message. |

#### MessageReactions

An interface that represents the reactions on a message.

It has the following properties:

| Property | Description | Type |
| -------- | ----------- | ---- |
| `[reactionType: string]` | A map where keys are reaction types (e.g., '👍', '❤️') and values contain reaction details. | [`Reaction`](#reaction)[] |

#### Reaction

An interface that represents a single reaction.

It has the following properties:

| Property | Description | Type |
| -------- | ----------- | ---- |
| `type` | The type of reaction (e.g., emoji). | String |
| `clientId` | The clientId of the user who added the reaction. | String |
| `createdAt` | The timestamp when the reaction was created. | Date |
| `metadata` | Optional metadata associated with the reaction. | [`ReactionMetadata`](#reactionmetadata) |
| `headers` | Optional headers associated with the reaction. | [`ReactionHeaders`](#reactionheaders) |

#### ReactionMetadata

Metadata associated with a reaction.

`ReactionMetadata: Record<string, unknown>`

#### ReactionHeaders

Headers associated with a reaction.

`ReactionHeaders: Record<string, number | string | boolean | null | undefined>`

### Returns

Returns a [`Subscription`](#subscription) object with an `unsubscribe()` method to remove the listener.

#### Subscription

A response object that allows you to control the subscription.

It has the following properties:

| Property | Description | Type |
| -------- | ----------- | ---- |
| `unsubscribe` | A method to unsubscribe from message reaction events. | `() => Promise<void>` |

<Code>
```javascript
const subscription = message.reactions.subscribe((event) => {
  console.log('Reaction event:', event.type);
  console.log('Message ID:', event.messageId);
  console.log('Reactions:', event.reactions);
});

// Unsubscribe when done
await subscription.unsubscribe();
```
</Code>

## Subscribe to raw message reaction events <a id="subscribeToRaw"/>

`message.reactions.subscribeToRaw()`

Subscribe to raw reaction events for this message. Raw events provide individual reaction additions and removals rather than aggregated summaries.

`subscribeToRaw(listener: MessageRawReactionListener): Subscription`

### Parameters

Use the following parameters:

| Parameter | Description | Type |
| --------- | ----------- | ---- |
| `listener` | Callback that will be called when a raw message reaction event is received. | [`MessageRawReactionListener`](#messagerawreactionlistener) |

#### MessageRawReactionListener

A listener that listens for raw message reaction events.

`MessageRawReactionListener: (event: MessageReactionRawEvent) => void`

It uses the following parameters:

| Parameter | Description | Type |
| --------- | ----------- | ---- |
| `event` | A raw message reaction event. | [`MessageReactionRawEvent`](#messagereactionrawevent) |

#### MessageReactionRawEvent

An interface that represents a raw message reaction event.

It has the following properties:

| Property | Description | Type |
| -------- | ----------- | ---- |
| `type` | The type of the reaction event. | [`MessageReactionEventType`](#messagereactioneventtype) |
| `messageId` | The ID of the message that the reaction is for. | String |
| `reaction` | The reaction that was added or removed. | [`Reaction`](#reaction) |

### Returns

Returns a [`Subscription`](#subscription) object with an `unsubscribe()` method to remove the listener.

<Code>
```javascript
const subscription = message.reactions.subscribeToRaw((event) => {
  if (event.type === 'ReactionAdded') {
    console.log(`${event.reaction.clientId} added ${event.reaction.type} to message ${event.messageId}`);
  } else if (event.type === 'ReactionRemoved') {
    console.log(`${event.reaction.clientId} removed ${event.reaction.type} from message ${event.messageId}`);
  }
});

// Unsubscribe when done
await subscription.unsubscribe();
```
</Code>
