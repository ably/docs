<Code>
```shell
ably push devices list [flags]
```
</Code>

Use the `ably push devices list` command to list push device registrations.

## Flags

You can optionally pass the following flags to the `ably push devices list` command:

### `--device-id` <a id="device-id"/>

Filter registrations by device ID.

### `--client-id` <a id="client-id"/>

Filter registrations by client ID.

### `--state` <a id="state"/>

Filter registrations by state. Options are `ACTIVE`, `FAILING`, and `FAILED`.

### `--limit` <a id="limit"/>

The maximum number of registrations to retrieve. Defaults to `100`.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

List all device registrations:

<Code>
```shell
ably push devices list
```
</Code>

Filter by state:

<Code>
```shell
ably push devices list --state ACTIVE
```
</Code>
