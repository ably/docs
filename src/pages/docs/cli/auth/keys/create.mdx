Use the `ably auth keys create` command to create a new API key for an app.

## Synopsis

<Code>
```shell
ably auth keys create --name <name> [flags]
```
</Code>

## Flags

### `--name` <a id="name"/> <RequiredBadge />

The name for the new API key.

### `--app` <a id="app"/>

The app ID to create the key for. Uses the currently selected app if not specified.

### `--capabilities` <a id="capabilities"/>

A JSON string specifying the [capabilities](/docs/auth/capabilities) for the key. Defaults to `'{"*":["*"]}'`.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — JSON output. Mutually exclusive flags.
* [`--verbose`](/docs/cli#verbose) — Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Create a basic API key:

<Code>
```shell
ably auth keys create --name "My API Key"
```
</Code>

Create a key with specific capabilities:

<Code>
```shell
ably auth keys create --name "Publish Only" --capabilities '{"my-channel":["publish"]}'
```
</Code>

Create an API key for a specific app:

<Code>
```shell
ably auth keys create --name "My New Key" --app APP_ID
```
</Code>

Create a key with wildcard capabilities:

<Code>
```shell
ably auth keys create --name "My New Key" --capabilities '{"*":["*"]}'
```
</Code>

Create a key with capabilities for multiple channels:

<Code>
```shell
ably auth keys create --name "My New Key" --capabilities '{"channel1":["publish","subscribe"],"channel2":["history"]}'
```
</Code>

Create a key with namespace-scoped publish capability:

<Code>
```shell
ably auth keys create --app APP_ID --name "MyKey" --capabilities '{"channel:*":["publish"]}'
```
</Code>

Create a key with mixed capabilities for a specific app:

<Code>
```shell
ably auth keys create --app APP_ID --name "MyOtherKey" --capabilities '{"channel:chat-*":["subscribe"],"channel:updates":["publish"]}'
```
</Code>

Create a key and output the result in JSON format:

<Code>
```shell
ably auth keys create --name "My New Key" --json
```
</Code>

Create a key and output the result in formatted JSON:

<Code>
```shell
ably auth keys create --name "My New Key" --pretty-json
```
</Code>
