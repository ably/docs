# Room reactions with Chat in Typescript

This folder contains the code for the Room reactions (Typescript) - a demo of how you can leverage [Ably Chat](https://ably.com/docs/products/chat) to reactions send to the room by the users.

## Getting started

First, rename `.env.example` to `.env.local` and fill in your Ably API KEY in the `VITE_PUBLIC_ABLY_KEY` environment variable.

Install dependencies:

```bash
yarn install
```

Run the development server:

```bash
yarn dev
```

Open [http://localhost:5173/](http://localhost:5173/) with your browser to see the result.
