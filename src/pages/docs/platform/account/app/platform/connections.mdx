The Connections section in your Ably dashboard provides comprehensive monitoring and management tools for all active client connections across your application. This interface enables you to track connection performance, troubleshoot client issues, and analyze connection patterns in realtime.

## Connection browser

The main connections interface displays all active connections with filtering and search capabilities to help you quickly locate specific connections.

### Connection finder and filtering

Use the connection finder to search for specific connections by Connection ID or Client ID. The interface also provides filtering options:

| Filter | Purpose |
|---|---|
| Connection finder | Search box to filter by connection ID or client ID |
| Country filter | Filter connections by geographic location |
| SDK filter | Filter connections by client SDK version |

The interface shows connection totals, giving you immediate awareness of active connections and helping you track connection volume over time.

### Connection list

Each connection entry displays essential information for rapid assessment:

| Column | Description | Purpose |
|---|---|---|
| Connection ID | Unique identifier for each client connection | Track and reference specific connections |
| Client ID | Application-defined client identifier or blank | Identify clients in your application logic |
| Country | Geographic location with flag indicator | Monitor regional usage patterns |
| SDK | Client SDK version and platform icon | Debug compatibility and version issues |
| Inspect button | Eye icon for detailed connection access | Access comprehensive connection monitoring |

Click the **inspect** button (eye icon) next to any connection to access detailed connection information and monitoring tools.

## Connection inspection

When you click the **inspect** button on any connection, you access a comprehensive connection monitoring interface that provides detailed visibility into that specific connection's behavior and configuration.

### Connection overview

The inspection page opens with a detailed overview showing critical connection information and realtime status. The header displays the full Connection ID with a copy button for easy reference, along with breadcrumb navigation back to the main connections list.

#### Connection details

The overview displays essential connection information:

| Detail | Description | Purpose |
|---|---|---|
| Client ID | Application-defined identifier or • N/A • | Identify clients in your application logic |
| Transport type | Connection method such as WebSocket | Understand connection protocol |
| Created | Connection establishment timestamp | Track connection duration |
| Status | Current connection state with indicator | Monitor connection health |
| IP address | Geographic location and IP address | Debug regional connectivity |
| SDK | Client library version and platform | Debug compatibility issues |
| Platform | Operating system or runtime environment | Environment-specific troubleshooting |

#### Authentication section

The authentication section shows the security configuration for this connection:

| Authentication detail | Description | Purpose |
|---|---|---|
| Auth type | Authentication method used, for example Basic | Verify authentication approach |
| Security | Security protocol in use, for example TLS | Confirm encryption status |
| Key ID | API key identifier used for authentication | Track API key usage |

#### Capabilities

The capabilities section displays feature support for this connection, shown as capability strings that indicate which Ably features the client can use.

#### Activity monitoring

The activity section provides realtime metrics with message flow visualization:

| Activity metric | Description | Purpose |
|---|---|---|
| Inbound message rate | Messages per second sent by this client | Monitor client activity |
| Outbound message rate | Messages per second delivered to this client | Track message delivery |

The activity chart visualizes message flow over time with separate tracking for inbound and outbound messages, helping you identify usage patterns and potential issues.

#### Live logs

The live logs section shows realtime events specific to this connection:

| Log detail | Description | Purpose |
|---|---|---|
| Timestamp | When each event occurred | Track event chronology |
| Resource ID | Which channel or resource was involved | Identify affected resources |
| Event type | Type of connection event | Categorize events |
| Level | Severity indicator with status badges | Prioritize troubleshooting |

## Attached channels

When you select the "Attached channels" tab from the connection inspection interface, you can view all channels that this specific connection is currently attached to.

### Channel activity monitoring

The attached channels view displays:

| Information | Purpose |
|---|---|
| Channel finder | Search box to filter channels by name |
| Channel count | Shows total attached channels |
| Channel list | Each channel with its activity metrics |

### Individual channel details

For each attached channel, the interface shows:

| Channel detail | Description | Purpose |
|---|---|---|
| Channel name | Full channel identifier | Identify specific channels |
| Modes | Connection modes like presence, publish, subscribe | Understand channel usage patterns |
| Inbound message rate | Messages per second sent by this client to the channel | Monitor client publishing activity |
| Outbound message rate | Messages per second delivered to this client from the channel | Track message delivery from channel |
| Activity chart | Realtime message flow visualization | Identify usage patterns and issues |
| Inspect button | Eye icon for detailed channel access | Navigate to channel inspection |

### Channel inspection navigation

When you click the **Inspect** button on any channel in the attached channels list, you navigate to the comprehensive channel inspection interface in the Pub/Sub section. This provides detailed channel monitoring including:

- Channel overview and analytics
- All connections to that channel
- Message history and realtime flow
- Presence tracking
- Channel configuration and rules

This seamless navigation allows you to move from connection-specific monitoring to channel-wide analysis, enabling comprehensive troubleshooting across both connection and channel perspectives.
