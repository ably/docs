<Code>
```shell
ably apps stats [ID] [flags]
```
</Code>

Use the `ably apps stats` command to retrieve detailed usage statistics for an Ably app. This provides metrics on connections, channels, messages, API requests, and more, with options for historical data and live updates.

## Arguments

Pass the following arguments to the `ably apps stats` command:

### `[ID]` <a id="id"/>

The App ID to get statistics for, for example: `Ab1CdE`.

If not provided, uses the current app from your CLI configuration.

## Flags

You can optionally pass the following flags to the `ably apps stats` command:

### `--start` <a id="start"/>

Start time in milliseconds since epoch for historical stats retrieval.

### `--end` <a id="end"/>

End time in milliseconds since epoch for historical stats retrieval.

### `--unit` <a id="unit"/>

Time unit for statistics aggregation. Options are `minute`, `hour`, `day`, or `month`. The default is `minute`.

### `--limit` <a id="limit"/>

Maximum number of statistics records to return. The default is `10`.

### `--live` <a id="live"/>

Subscribe to live statistics updates using minute intervals.

### `--interval` <a id="interval"/>

Polling interval in seconds when using `--live` mode. The default is `6` seconds.

### `--debug` <a id="debug"/>

Show debug information for live statistics polling.

### `--help | -h` <a id="help"/>

Show CLI help for this command.

## Examples

Get recent app statistics:

<Code>
```shell
ably apps stats
```
</Code>

Get statistics for a specific app:

<Code>
```shell
ably apps stats app-id
```
</Code>

Get hourly statistics:

<Code>
```shell
ably apps stats --unit hour
```
</Code>

Get statistics for a specific time range:

<Code>
```shell
ably apps stats app-id --start 1618005600000 --end 1618091999999
```
</Code>

Subscribe to live statistics updates:

<Code>
```shell
ably apps stats --live
```
</Code>
