<Code>
```shell
ably rooms list [flags]
```
</Code>

Use the `ably rooms list` command to list the active chat rooms in your app.

## Flags

You can optionally pass the following flags to the `ably rooms list` command:

### `--limit` <a id="limit"/>

Maximum number of rooms to return. The default is `100`. The maximum is `1000`.

### `--prefix | -p` <a id="prefix"/>

Filter rooms by prefix. Only rooms whose names start with the specified prefix will be returned.

## Global flags

The following global flags are supported using the `ably rooms list` command:

* [`--token`](/docs/cli#token)
* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--api-key`](/docs/cli#api-key)
* [`--access-token`](/docs/cli#access-token)
* [`--client-id`](/docs/cli#client-id)
* [`--env`](/docs/cli#env)
* [`--host`](/docs/cli#host)
* [`--endpoint`](/docs/cli#endpoint)
* [`--verbose`](/docs/cli#verbose)

## Examples

### List all active chat rooms

Get a list of all currently active chat rooms:

<Code>
```shell
ably rooms list
```
</Code>

**Output:**
<Code>
```text
Using: App=example-app (Ab1CdE) â€¢ Key=example-key (Ab1CdE.FgHiJk)

No active chat rooms found.
```
</Code>

### List rooms with a limit

Get only the first 5 active chat rooms:

<Code>
```shell
ably rooms list --limit 5
```
</Code>

### Filter rooms by prefix

List only rooms that start with "game-":

<Code>
```shell
ably rooms list --prefix "game-"
```
</Code>

### Filter with prefix and limit

Get the first 3 rooms starting with "chat-":

<Code>
```shell
ably rooms list --prefix "chat-" --limit 3
```
</Code>
