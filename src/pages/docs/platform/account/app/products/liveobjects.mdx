The LiveObjects section of the [app dashboard](https://ably.com/accounts/any/apps/any/liveobjects) provides tools for monitoring and managing synchronized state objects across your application. From this page, you can access getting started guides for different platforms and explore ready-to-use examples.

## CLI <a id="cli"/>

The [Ably CLI](/docs/platform/tools/cli) provides command-line tools for managing LiveObjects operations, testing state synchronization, and automating workflows.
