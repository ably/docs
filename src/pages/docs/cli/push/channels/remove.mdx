<Code>
```shell
ably push channels remove --channel <channel> [flags]
```
</Code>

Use the `ably push channels remove` command to remove a push channel subscription.

## Flags

You can optionally pass the following flags to the `ably push channels remove` command:

### `--channel` <a id="channel"/>

The channel to remove the subscription from. Required.

### `--device-id` <a id="device-id"/>

The device ID to unsubscribe. Exclusive with `--client-id`.

### `--client-id` <a id="client-id"/>

The client ID to unsubscribe. Exclusive with `--device-id`.

### `--force | -f` <a id="force"/>

Skip confirmation prompt. Required when using `--json`. The default is `false`.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Remove a device subscription from a channel:

<Code>
```shell
ably push channels remove --channel my-channel --device-id "device123"
```
</Code>

Remove a client subscription from a channel:

<Code>
```shell
ably push channels remove --channel my-channel --client-id "user456"
```
</Code>
