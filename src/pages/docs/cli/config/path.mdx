Use the `ably config path` command to print the path to the Ably CLI config file.

## Synopsis

<Code>
```shell
ably config path
```
</Code>

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Print the config file path:

<Code>
```shell
ably config path
```
</Code>

Open the config file in VS Code:

<Code>
```shell
code $(ably config path)
```
</Code>

Open the config file in Vim:

<Code>
```shell
vim $(ably config path)
```
</Code>

Print the config file path in JSON format:

<Code>
```shell
ably config path --json
```
</Code>
