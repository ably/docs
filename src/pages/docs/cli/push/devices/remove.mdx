Use the `ably push devices remove` command to remove a push device registration.

## Synopsis

<Code>
```shell
ably push devices remove <device-id> [flags]
```
</Code>

## Arguments

### `device-id` <a id="device-id"/> <RequiredBadge />

The ID of the device to remove.

## Flags

### `--force | -f` <a id="force"/>

Skip confirmation prompt. Required when using `--json`. The default is `false`.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — JSON output. Mutually exclusive flags.
* [`--verbose`](/docs/cli#verbose) — Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Remove a device registration:

<Code>
```shell
ably push devices remove "device123"
```
</Code>

Force remove without confirmation:

<Code>
```shell
ably push devices remove "device123" --force
```
</Code>

Remove a device registration and output in JSON format:

<Code>
```shell
ably push devices remove device-123 --json
```
</Code>
