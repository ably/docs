Use the `ably rooms messages reactions send` command to send a reaction to a message in a chat room.

## Synopsis

<Code>
```shell
ably rooms messages reactions send <room> <message-serial> <reaction> [flags]
```
</Code>

## Arguments

### `room` <a id="room"/> <RequiredBadge />

The name of the chat room containing the message.

### `message-serial` <a id="message-serial"/> <RequiredBadge />

The serial identifier of the message to react to.

### `reaction` <a id="reaction"/> <RequiredBadge />

The emoji reaction to send.

## Flags

### `--type` <a id="type"/>

The type of reaction. Options are `unique`, `distinct`, or `multiple`.

### `--count` <a id="count"/>

The number of reactions to send. Only applicable when `--type` is set to `multiple`.

### `--client-id` <a id="client-id"/>

A client ID to use when sending the reaction.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — JSON output. Mutually exclusive flags.
* [`--verbose`](/docs/cli#verbose) — Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Send a reaction to a message:

<Code>
```shell
ably rooms messages reactions send my-room "01234567890:0" "thumbsup"
```
</Code>

Send a reaction with a specific type:

<Code>
```shell
ably rooms messages reactions send my-room "01234567890:0" "heart" --type unique
```
</Code>

Send multiple reactions:

<Code>
```shell
ably rooms messages reactions send my-room "01234567890:0" "clap" --type multiple --count 5
```
</Code>

Send a reaction using an API key environment variable:

<Code>
```shell
ABLY_API_KEY="YOUR_API_KEY" ably rooms messages reactions send my-room message-serial heart
```
</Code>

Send a reaction and output the result in JSON format:

<Code>
```shell
ably rooms messages reactions send my-room message-serial thumbsup --json
```
</Code>
