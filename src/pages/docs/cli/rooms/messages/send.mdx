<Code>
```shell
ably rooms messages send <room> <text> [flags]
```
</Code>

Use the `ably rooms messages send` command to send a message to an Ably Chat room.

Use another terminal instance and the [`ably rooms messages subscribe`](/docs/cli/rooms/messages/subscribe) command to receive the sent messages.

## Arguments

Pass the following arguments to the `ably rooms messages send` command:

### `<room>` <a id="room"/>

The room to send the message to, for example: `example-room`.

### `<text>` <a id="text"/>

The message text to send, for example: `"Hello everyone!"`.

## Flags

You can optionally pass the following flags to the `ably rooms messages send` command:

### `--count | -c` <a id="count"/>

The number of messages to send. The default is `1`.

### `--delay | -d` <a id="delay"/>

The delay between messages being sent, in milliseconds. The default is `40`.

### `--metadata` <a id="metadata"/>

Additional metadata for the message in JSON format, for example: `{"priority":"high","category":"announcement"}`.

## Global flags

The following global flags are supported using the `ably rooms messages send` command:

* [`--token`](/docs/cli#token)
* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--api-key`](/docs/cli#api-key)
* [`--access-token`](/docs/cli#access-token)
* [`--client-id`](/docs/cli#client-id)
* [`--env`](/docs/cli#env)
* [`--host`](/docs/cli#host)
* [`--endpoint`](/docs/cli#endpoint)
* [`--verbose`](/docs/cli#verbose)

## Examples

### Send a simple message

Send a basic message to a chat room:

<Code>
```shell
ably rooms messages send example-room "Hello everyone!"
```
</Code>

### Send multiple messages

Send 3 messages with a delay between them:

<Code>
```shell
ably rooms messages send example-room "Countdown message" --count 3 --delay 1000
```
</Code>

### Send a message with metadata

Send a message with additional metadata:

<Code>
```shell
ably rooms messages send example-room "Important announcement" --metadata '{"priority":"high","category":"announcement"}'
```
</Code>
