<Code>
```shell
ably connections logs [flags]
```
</Code>

Use the `ably connections logs` command to subscribe to connection lifecycle logs. This command provides information about connection events such as connections opening, closing, and status changes.

<Aside data-type="note">
This command is an alias for `ably logs connection-lifecycle subscribe`.
</Aside>

For complete documentation on connection lifecycle logs, see [`ably logs connection-lifecycle subscribe`](/docs/cli/logs/connection-lifecycle/subscribe).

## Flags

You can optionally pass the following flags to the `ably connections logs` command:

### `--help | -h` <a id="help"/>

Show CLI help for this command.

The following global flags are supported using the `ably connections logs` command:

* [`--api-key`](/docs/cli#api-key)
* [`--access-token`](/docs/cli#access-token)
* [`--client-id`](/docs/cli#client-id)
* [`--endpoint`](/docs/cli#endpoint)
* [`--env`](/docs/cli#env)
* [`--host`](/docs/cli#host)
* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
