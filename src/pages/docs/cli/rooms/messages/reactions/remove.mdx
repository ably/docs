<Code>
```shell
ably rooms messages reactions remove <room> <message-serial> <reaction> [flags]
```
</Code>

Use the `ably rooms messages reactions remove` command to remove a reaction from a message in a chat room.

## Arguments

Pass the following arguments to the `ably rooms messages reactions remove` command:

### `<room>` <a id="room"/>

The name of the chat room containing the message.

### `<message-serial>` <a id="message-serial"/>

The serial identifier of the message to remove the reaction from.

### `<reaction>` <a id="reaction"/>

The emoji reaction to remove.

## Flags

You can optionally pass the following flags to the `ably rooms messages reactions remove` command:

### `--type` <a id="type"/>

The type of reaction to remove. Options are `unique`, `distinct`, or `multiple`.

### `--client-id` <a id="client-id"/>

A client ID to use when removing the reaction.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Remove a reaction from a message:

<Code>
```shell
ably rooms messages reactions remove my-room "01a2b3c4d5@1234567890000-0" "thumbsup"
```
</Code>

Remove a reaction with a specific type:

<Code>
```shell
ably rooms messages reactions remove my-room "01a2b3c4d5@1234567890000-0" "heart" --type unique
```
</Code>
