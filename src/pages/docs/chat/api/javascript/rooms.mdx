The `Rooms` interface manages the lifecycle of chat rooms.

Access it via the `rooms` property on a [`ChatClient`](/docs/chat/api/javascript/chat-client) instance.

<Code>
```javascript
const rooms = chatClient.rooms;
```
</Code>

## Create a room <a id="get"/>

`rooms.get(name, options)`

Create or retrieve a [`Room`](/docs/chat/api/javascript/room) instance. Optionally provide custom configuration to the room.

Call [`release()`](#release) when the `Room` is no longer needed. If a call to `get()` is made for a room that is currently being released, then the promise will only resolve when the release operation is complete.

If a call to `get()` is made, followed by a subsequent call to `release()` before the promise resolves, then the promise will reject with an error.

<Code>
```javascript
const options = {
  typing: {
    heartbeatThrottleMs: 5000,
  },
  presence: {
    enableEvents: true,
  },
  occupancy: {
    enableEvents: true,
  },
  messages: {
    rawMessageReactions: false,
  },
};

const room = await chatClient.rooms.get('basketball-stream', options);

// When finished with the room
await chatClient.rooms.release('basketball-stream');
```
</Code>

### Returns

`Promise<Room>`

A promise that resolves to a [`Room`](/docs/chat/api/javascript/room) instance.

If the room already exists with different options, the promise rejects with an error. If [`release()`](#release) is called before `get()` resolves, the promise also rejects.

## Release a room <a id="release"/>

`rooms.release(name)`

Releases a room, freeing its resources and detaching it from Ably.

After release, the room object is no longer usable. To use the room again, call [`get()`](#get) to create a new instance. This method only releases the reference and detaches from Ably, it doesn't unsubscribe existing event listeners.

<Code>
```javascript
await chatClient.rooms.release('my-room');
```
</Code>

### Parameters

| Parameter | Description | Type |
| --- | --- | --- |
| name | The unique identifier of the room to release. | string |

### Returns

`Promise<void>`

A promise that resolves when the room is fully released.

## Dispose of all rooms <a id="dispose"/>

`rooms.dispose()`

Releases all rooms, freeing their resources and detaching them from Ably. After calling this method, any existing room objects obtained from this instance are no longer usable.

<Code>
```javascript
// Release all rooms at once
await chatClient.rooms.dispose();
```
</Code>

### Returns

`Promise<void>`

A promise that resolves when all rooms have been released.
