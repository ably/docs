Use the `ably spaces locations set` command to set location in a space.

## Synopsis

<Code>
```shell
ably spaces locations set <space-name> --location <json> [options]
```
</Code>

## Arguments

### `space-name` <a id="space-name"/> <RequiredBadge />

The name of the space to set the location in.

## Options

### `--location` <a id="location"/> <RequiredBadge />

JSON location data to set.

### `--duration | -D` <a id="duration"/>

The duration in seconds to maintain the location before automatically removing it.

### `--client-id` <a id="client-id"/>

A client ID to use when setting the location.

### `--json` <a id="json"/>

Output results as compact JSON. Mutually exclusive with `--pretty-json`.

### `--pretty-json` <a id="pretty-json"/>

Output results in formatted JSON. Mutually exclusive with `--json`.

### `--verbose | -v` <a id="verbose"/>

Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Set a location in a space:

<Code>
```shell
ably spaces locations set my-space --location '{"page": "sheet-1", "cell": "A1"}'
```
</Code>

Set a location with a specific duration:

<Code>
```shell
ably spaces locations set my-space --location '{"slide": 5}' --duration 120
```
</Code>

Set a location and output in JSON format:

<Code>
```shell
ably spaces locations set my-space --location '{"x":10,"y":20}' --json
```
</Code>
