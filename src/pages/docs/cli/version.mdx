Use the `ably version` command to display version information for the Ably CLI, including the CLI version, platform architecture, and Node.js version.

## Synopsis

<Code>
```shell
ably version
```
</Code>

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — JSON output. Mutually exclusive flags.
* [`--verbose`](/docs/cli#verbose) — Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Display the CLI version:

<Code>
```shell
ably version
```
</Code>
