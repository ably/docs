Use the `ably push channels save` command to subscribe a device or client to push notifications on a channel.

## Synopsis

<Code>
```shell
ably push channels save --channel <channel> [flags]
```
</Code>

## Flags

### `--channel` <a id="channel"/> `(Required)`

The channel to subscribe to.

### `--device-id` <a id="device-id"/>

The device ID to subscribe. Exclusive with `--client-id`.

### `--client-id` <a id="client-id"/>

The client ID to subscribe. Exclusive with `--device-id`.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Subscribe a device to push notifications on a channel:

<Code>
```shell
ably push channels save --channel my-channel --device-id "device123"
```
</Code>

Subscribe a client to push notifications on a channel:

<Code>
```shell
ably push channels save --channel my-channel --client-id "user456"
```
</Code>

Save a push channel subscription and output in JSON format:

<Code>
```shell
ably push channels save --channel my-channel --device-id device-123 --json
```
</Code>
