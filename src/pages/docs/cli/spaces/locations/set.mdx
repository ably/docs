<Code>
```shell
ably spaces locations set <name> --location <json> [flags]
```
</Code>

Use the `ably spaces locations set` command to set location in a space.

## Arguments

Pass the following arguments to the `ably spaces locations set` command:

### `<name>` <a id="name"/>

The name of the space to set the location in.

## Flags

You can optionally pass the following flags to the `ably spaces locations set` command:

### `--location` <a id="location"/>

JSON location data to set. This flag is required.

### `--duration | -D` <a id="duration"/>

The duration in seconds to maintain the location before automatically removing it.

### `--client-id` <a id="client-id"/>

A client ID to use when setting the location.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Set a location in a space:

<Code>
```shell
ably spaces locations set my-space --location '{"page": "sheet-1", "cell": "A1"}'
```
</Code>

Set a location with a specific duration:

<Code>
```shell
ably spaces locations set my-space --location '{"slide": 5}' --duration 120
```
</Code>
