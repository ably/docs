Access your [account](https://ably.com/login) dashboard to manage organizations, apps, team members, billing, usage monitoring, and reports.

## Organization and account selection <a id="organization-and-account-selection"/>

The dashboard includes an organization and account selector that allows you to navigate between different contexts:

| Element | Description |
|---------|-------------|
| Organization | Switch between different organizations you belong to, each potentially containing multiple accounts |
| Standalone accounts | Access individual accounts that aren't part of an organization structure |
| Account context | Select the specific account you want to manage, with each account displaying its associated pricing plan (Free, Enterprise, etc.) |

## Account listing <a id="account-listing"/>

The accounts dropdown displays all available accounts within your selected organization:

| Element | Description |
|---------|-------------|
| Account name | Shows individual account names (Ably, Ably Coco, Dev env, Docs test) with their current status |
| Plan type | Displays the pricing plan for each account (Free, Enterprise) |
| View all accounts | Provides access to see the complete list of accounts when you have multiple accounts |

## Dashboard tools <a id="dashboard-tools"/>

The dashboard header provides quick access to helpful tools:

| Tool | Description |
|------|-------------|
| Ask AI | Access AI-powered assistance for questions about Ably features, documentation, and troubleshooting |
| Feedback | Provide feedback on the dashboard experience or report issues directly to the Ably team |
| Help | Access help documentation and support resources |
| CLI | Open the command-line interface for testing and running dashboard functionality programmatically |

## User profile menu <a id="user-profile-menu"/>

The user profile dropdown provides access to personal account management:

| Option | Description |
|--------|-------------|
| Profile settings | Access and modify your personal account information, preferences, and security settings |
| Personal access tokens | Create and manage personal access tokens for API authentication and development |
| Sidebar behavior | Customize dashboard sidebar display preferences and navigation behavior |
| Logout | Sign out of your Ably account securely |

## Apps <a id="apps-overview"/>

The Apps section provides a centralized view of all your applications within the selected account:

| Element | Description |
|---------|-------------|
| App cards | Display individual app information including name, message counts, and connection statistics |
| Usage metrics | Show last 24-hour activity with visual charts for messages and connections |
| Quick actions | Access "View App" links to navigate directly to each app's detailed dashboard |
| Create new app | Easily create additional applications from the main Apps interface |

## Team <a id="team"/>

Manage team members and their roles within your account:

| Element | Description |
|---------|-------------|
| User information | View team member details including name, email address, and account owner status |
| Role assignment | Assign and manage multiple roles per user (Admin, Developer, Billing) with checkbox selection |
| User management | Invite new users to the account and remove existing team members as needed |
| 2FA status | View which users have two-factor authentication enabled for enhanced security |


## Usage <a id="usage"/>

Monitor your account's resource consumption with detailed usage statistics:

* The usage statistics table monitors app performance by tracking resource usage, such as messages, data transferred, connections, and channels.
* The statistics chart visualizes app usage data over time, allowing users to define specific time ranges, zoom in on different periods, and analyze metrics in detail.

## Limits <a id="limits"/>

Understand and manage your package [limits](/docs/platform/pricing/limits) by tracking usage across messages, data, connections, and channels:

* Limits are applied to prevent service disruption and vary by package.
* Limits can be time-based, quantity-based, or rate-based, and they may apply either per connection/channel or across the entire account.
* Notifications are provided when limits are nearing or exceeded, and detailed logs of warnings and exceeded limits are accessible in the account dashboard.

## Reports <a id="reports"/>

Access powerful reporting tools to analyze your account's realtime activity and consumption patterns:

| Report type | Description |
|-------------|-------------|
| Global usage | Consolidated view of your account's realtime activity and efficiency, showing growth trends, traffic breakdowns, and billable consumption patterns |
| Global usage by app | Consolidated view of your account's realtime activity, showing app-level breakdowns and billable consumption patterns |
