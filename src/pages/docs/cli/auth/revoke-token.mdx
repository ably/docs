Use the `ably auth revoke-token` command to revoke an Ably token.

## Synopsis

<Code>
```shell
ably auth revoke-token <token> [options]
```
</Code>

## Arguments

### `token` <a id="token"/> <RequiredBadge />

The token to revoke.

## Options

### `--app` <a id="app"/>

The app ID the token belongs to. Uses the currently selected app if not specified.

### `--client-id | -c` <a id="client-id"/>

The client ID to revoke tokens for.

### `--json` <a id="json"/>

Output results as compact JSON. Mutually exclusive with `--pretty-json`.

### `--pretty-json` <a id="pretty-json"/>

Output results in formatted JSON. Mutually exclusive with `--json`.

### `--verbose | -v` <a id="verbose"/>

Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Revoke a specific token:

<Code>
```shell
ably auth revoke-token "xVLyHw.DRE6hQ..."
```
</Code>

Revoke tokens for a specific client ID:

<Code>
```shell
ably auth revoke-token "xVLyHw.DRE6hQ..." --client-id "user-123"
```
</Code>

Revoke a token and output the result in JSON format:

<Code>
```shell
ably auth revoke-token TOKEN --json
```
</Code>

Revoke a token and output the result in formatted JSON:

<Code>
```shell
ably auth revoke-token TOKEN --pretty-json
```
</Code>
