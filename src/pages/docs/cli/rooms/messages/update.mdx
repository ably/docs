<Code>
```shell
ably rooms messages update <room> <serial> <text> [flags]
```
</Code>

Use the `ably rooms messages update` command to update a message in an Ably Chat room.

## Arguments

Pass the following arguments to the `ably rooms messages update` command:

### `<room>` <a id="room"/>

The name of the chat room containing the message.

### `<serial>` <a id="serial"/>

The serial identifier of the message to update.

### `<text>` <a id="text"/>

The new text content for the message.

## Flags

You can optionally pass the following flags to the `ably rooms messages update` command:

### `--description` <a id="description"/>

A description of the update.

### `--headers` <a id="headers"/>

A JSON string of headers to attach to the update.

### `--metadata` <a id="metadata"/>

A JSON string of metadata to attach to the updated message.

### `--client-id` <a id="client-id"/>

A client ID to use when updating the message.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Update a message:

<Code>
```shell
ably rooms messages update my-room "01a2b3c4d5@1234567890000-0" "Updated message text"
```
</Code>

Update a message with a description:

<Code>
```shell
ably rooms messages update my-room "01a2b3c4d5@1234567890000-0" "Fixed typo" --description "Corrected spelling"
```
</Code>

Update a message with metadata:

<Code>
```shell
ably rooms messages update my-room "01a2b3c4d5@1234567890000-0" "Edited message" --metadata '{"edited": true}'
```
</Code>
