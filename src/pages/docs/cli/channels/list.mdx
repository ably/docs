Use the `ably channels list` command to list active channels using the channel enumeration API.

## Synopsis

<Code>
```shell
ably channels list [flags]
```
</Code>

## Flags

### `--limit` <a id="limit"/>

The maximum number of channels to return. Defaults to `100`.

### `--prefix | -p` <a id="prefix"/>

Filter channels by a name prefix.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

List all active channels:

<Code>
```shell
ably channels list
```
</Code>

List channels with a specific prefix:

<Code>
```shell
ably channels list --prefix "chat:"
```
</Code>

List channels with a limit:

<Code>
```shell
ably channels list --limit 10
```
</Code>
