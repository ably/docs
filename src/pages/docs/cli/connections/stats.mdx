<Code>
```shell
ably connections stats [flags]
```
</Code>

Use the `ably connections stats` command to view detailed [connection statistics](/docs/connections) for your Ably app. This provides metrics on connection activity, including connection counts, rates, and trends over time.

## Flags

You can optionally pass the following flags to the `ably connections stats` command:

### `--start` <a id="start"/>

Start time in milliseconds since epoch for historical stats retrieval.

### `--end` <a id="end"/>

End time in milliseconds since epoch for historical stats retrieval.

### `--unit` <a id="unit"/>

Time unit for statistics aggregation. Options are `minute`, `hour`, `day`, or `month`. The default is `minute`.

### `--limit` <a id="limit"/>

Maximum number of statistics records to return. The default is `10`.

### `--live` <a id="live"/>

Subscribe to live statistics updates using minute intervals.

### `--interval` <a id="interval"/>

Polling interval in seconds when using `--live` mode. The default is `6` seconds.

### `--debug` <a id="debug"/>

Show debug information for live statistics polling.

### `--help | -h` <a id="help"/>

Show CLI help for this command.

## Global flags

The following global flags are supported using the `ably connections stats` command:

* [`--api-key`](/docs/cli#api-key)
* [`--access-token`](/docs/cli#access-token)
* [`--client-id`](/docs/cli#client-id)
* [`--endpoint`](/docs/cli#endpoint)
* [`--env`](/docs/cli#env)
* [`--host`](/docs/cli#host)
* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--token`](/docs/cli#token)
* [`--verbose`](/docs/cli#verbose)

## Examples

Get recent connection statistics:

<Code>
```shell
ably connections stats
```
</Code>

Get hourly connection statistics:

<Code>
```shell
ably connections stats --unit hour
```
</Code>

Get connection statistics for a specific time range:

<Code>
```shell
ably connections stats --start 1618005600000 --end 1618091999999
```
</Code>

Subscribe to live connection statistics:

<Code>
```shell
ably connections stats --live
```
</Code>
