@ARTChannelEvent@ is the enum emitted as the event in @ARTRealtimeChannel.on@. The variants correspond to "@Realtime Channel@ state":/realtime/channels#channel-state changes, plus an arbitrary @Error@.

```[objc]
  typedef NS_ENUM(NSUInteger, ARTChannelEvent) {
      ARTChannelEventInitialized,
      ARTChannelEventAttaching,
      ARTChannelEventAttached,
      ARTChannelEventDetaching,
      ARTChannelEventDetached,
      ARTChannelEventFailed,
      ARTChannelEventError
  };
```

```[swift]
  enum ARTChannelEvent : UInt {
      case Initialized
      case Attaching
      case Attached
      case Detaching
      case Detached
      case Failed
      case Error
  }
```
