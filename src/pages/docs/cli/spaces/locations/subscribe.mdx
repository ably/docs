```shell
ably spaces locations subscribe <name> [flags]
```
</Code>

Use the `ably spaces locations subscribe` command to subscribe to location updates for members in a space.

## Arguments

Pass the following arguments to the `ably spaces locations subscribe` command:

### `<name>` <a id="name"/>

The name of the space to subscribe to location updates in.

## Flags

You can optionally pass the following flags to the `ably spaces locations subscribe` command:

### `--duration | -D` <a id="duration"/>

The duration in seconds to subscribe for before automatically unsubscribing.

### `--client-id` <a id="client-id"/>

A client ID to use when subscribing.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Subscribe to location updates in a space:

<Code>
```shell
ably spaces locations subscribe my-space
```
</Code>

Subscribe for a specific duration:

<Code>
```shell
ably spaces locations subscribe my-space --duration 60
```
</Code>
