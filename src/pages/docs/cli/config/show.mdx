---

<Code>
```shell
ably config show
```
</Code>

Use the `ably config show` command to display the contents of the Ably CLI config file.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Display the config file contents:

<Code>
```shell
ably config show
```
</Code>

Display the config file contents in JSON format:

<Code>
```shell
ably config show --json
```
</Code>
