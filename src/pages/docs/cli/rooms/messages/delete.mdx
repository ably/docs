Use the `ably rooms messages delete` command to delete a message in an Ably Chat room.

## Synopsis

<Code>
```shell
ably rooms messages delete <room> <serial> [flags]
```
</Code>

## Arguments

### `room` <a id="room"/> `(Required)`

The name of the chat room containing the message.

### `serial` <a id="serial"/> `(Required)`

The serial identifier of the message to delete.

## Flags

### `--description` <a id="description"/>

A description of the reason for deleting the message.

### `--client-id` <a id="client-id"/>

A client ID to use when deleting the message.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Delete a message:

<Code>
```shell
ably rooms messages delete my-room "01234567890:0"
```
</Code>

Delete a message with a description:

<Code>
```shell
ably rooms messages delete my-room "01234567890:0" --description "Removed inappropriate content"
```
</Code>

Delete a room message and output the result in JSON format:

<Code>
```shell
ably rooms messages delete my-room "serial-001" --json
```
</Code>
