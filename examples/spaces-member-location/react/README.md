# Member location example

This folder contains the code for the member location (React) - a demo of how you can leverage [Ably Spaces](https://github.com/ably/spaces) to track where members are in on a page.

## Getting started

First, in `/examples/spaces-member-location/react/` rename `.env.example` to `.env.local` and enter your Ably API KEY in the `NEXT_PUBLIC_ABLY_KEY` environment variable.

Change directory to `/examples/spaces-member-location/react/` and install dependencies:

```bash
yarn install
```

Run the development server:

```bash
yarn dev
```

Open two tabs to [http://localhost:3000](http://localhost:3000) with your browser to see the result.
