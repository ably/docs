@DataRequestQuery@ is a type that encapsulates the parameters needed for history and stats. Where the stats method uses @StatsDataRequestQuery@. For example usage see 
"@Channel#history@":/realtime/history#channel-history. 

h4.
  dotnet: Members

- Start? := The start of the queried interval. Default is null 
- End? := The end of the queried interval. Default is null
- Limit? := By default it is null. Limits the number of items returned by history or stats
- Direction := Enum which is either Forwards or Backwards. Defaults to Forwards
- ExtraParameters := Dictionary<string, string> it is for any extra query parameters that need to be passed. It is mainly used by the library when paging
