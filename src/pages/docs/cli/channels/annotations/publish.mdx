Use the `ably channels annotations publish` command to publish an annotation on a channel message.

## Synopsis

<Code>
```shell
ably channels annotations publish <channel> <serial> <type> [flags]
```
</Code>

## Arguments

### `channel` <a id="channel"/> <RequiredBadge />

The name of the channel containing the message.

### `serial` <a id="serial"/> <RequiredBadge />

The serial of the message to annotate.

### `type` <a id="type"/> <RequiredBadge />

The annotation type. For example, `reactions:unique.v1` or `metrics:total.v1`.

## Flags

### `--name | -n` <a id="name"/>

The annotation name. For example, an emoji name such as `thumbsup`.

### `--count` <a id="count"/>

The count value for the annotation. Used with `multiple.v1` type annotations.

### `--data` <a id="data"/>

A JSON payload to include with the annotation.

### `--encoding | -e` <a id="encoding"/>

The encoding of the annotation data.

### `--client-id` <a id="client-id"/>

A client ID to use when publishing the annotation.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — JSON output. Mutually exclusive flags.
* [`--verbose`](/docs/cli#verbose) — Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Publish a simple annotation:

<Code>
```shell
ably channels annotations publish my-channel "01234567890:0" "reactions:unique.v1"
```
</Code>

Publish a reaction with a name:

<Code>
```shell
ably channels annotations publish my-channel "01234567890:0" "reactions:unique.v1" --name thumbsup
```
</Code>

Publish an annotation with a count:

<Code>
```shell
ably channels annotations publish my-channel "01234567890:0" "metrics:multiple.v1" --name views --count 5
```
</Code>

Publish a total-type metric annotation:

<Code>
```shell
ably channels annotations publish my-channel "01234567890:0" "metrics:total.v1"
```
</Code>

Publish a flag annotation such as a read receipt:

<Code>
```shell
ably channels annotations publish my-channel "01234567890:0" "receipts:flag.v1"
```
</Code>

Publish an annotation and output the result in JSON format:

<Code>
```shell
ably channels annotations publish my-channel "01234567890:0" "metrics:total.v1" --json
```
</Code>
