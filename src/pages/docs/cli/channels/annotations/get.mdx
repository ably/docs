```shell
ably channels annotations get <channel> <serial> [flags]
```
</Code>

Use the `ably channels annotations get` command to get annotations for a channel message.

## Arguments

Pass the following arguments to the `ably channels annotations get` command:

### `<channel>` <a id="channel"/>

The name of the channel containing the message.

### `<serial>` <a id="serial"/>

The serial of the message to get annotations for.

## Flags

You can optionally pass the following flags to the `ably channels annotations get` command:

### `--limit` <a id="limit"/>

The maximum number of annotations to return. Defaults to `100`.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Get annotations for a message:

<Code>
```shell
ably channels annotations get my-channel "01a2b3c4d5@1234567890000-0"
```
</Code>

Get annotations with a limit:

<Code>
```shell
ably channels annotations get my-channel "01a2b3c4d5@1234567890000-0" --limit 10
```
</Code>
