Use the `ably apps rules delete` command to delete a [channel rule](/docs/channels#rules) from an Ably application.

## Synopsis

<Code>
```shell
ably apps rules delete <name-or-id> [options]
```
</Code>

## Arguments

### `name-or-id` <a id="name-or-id"/> <RequiredBadge />

The name or ID of the channel rule to delete.

## Options

### `--app` <a id="app"/>

The app ID to delete the rule from. Uses the currently selected app if not specified.

### `--force | -f` <a id="force"/>

Skip the confirmation prompt and delete the rule immediately. Required when using `--json`.

### `--json` <a id="json"/>

Output results as compact JSON. Mutually exclusive with `--pretty-json`.

### `--pretty-json` <a id="pretty-json"/>

Output results in formatted JSON. Mutually exclusive with `--json`.

### `--verbose | -v` <a id="verbose"/>

Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Delete a channel rule by name:

<Code>
```shell
ably apps rules delete "chat"
```
</Code>

Delete a channel rule without a confirmation prompt:

<Code>
```shell
ably apps rules delete "chat" --force
```
</Code>

Delete a channel rule for a specific app:

<Code>
```shell
ably apps rules delete events --app "My App"
```
</Code>

Delete a channel rule and output the result in JSON format:

<Code>
```shell
ably apps rules delete chat --json
```
</Code>

Delete a channel rule and output the result in formatted JSON:

<Code>
```shell
ably apps rules delete chat --pretty-json
```
</Code>
