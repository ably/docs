Use the `ably support ask` command to ask a question to the Ably AI agent for help.

## Synopsis

<Code>
```shell
ably support ask <question> [flags]
```
</Code>

## Arguments

### `question` <a id="question"/> `(Required)`

The question to ask the Ably AI agent.

## Flags

### `--continue` <a id="continue"/>

Continue a previous conversation with the AI agent. Defaults to `false`.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Ask a basic question:

<Code>
```shell
ably support ask "How do I create an app?"
```
</Code>

Ask a technical question:

<Code>
```shell
ably support ask "How do I authenticate with token auth?"
```
</Code>

Continue a previous conversation:

<Code>
```shell
ably support ask "Can you give me more details?" --continue
```
</Code>
