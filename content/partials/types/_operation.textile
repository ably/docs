An @Operation@ contains the details of an operation, such as update or deletion, supplied by the actioning client.

h4.
  default: Properties

- <span lang="default">clientId</span> := The client ID of the client that initiated the operation<br>__Type: @String@__

- <span lang="default">description</span> := The description provided by the client that initiated the operation<br>__Type: @String@__

- <span lang="default">metadata</span> := A JSON object of string key-value pairs that may contain metadata associated with the operation<br>__Type: @Record<String, String>@__
