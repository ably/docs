Use the `ably push channels` command group to manage push notification channel subscriptions.

## Subcommands

| Command | Description |
| ------- | ----------- |
| [`ably push channels list`](/docs/cli/push/channels/list) | List push channel subscriptions. |
| [`ably push channels list-channels`](/docs/cli/push/channels/list-channels) | List channels with push subscriptions. |
| [`ably push channels save`](/docs/cli/push/channels/save) | Subscribe a device or client to push notifications on a channel. |
| [`ably push channels remove`](/docs/cli/push/channels/remove) | Remove a push channel subscription. |
| [`ably push channels remove-where`](/docs/cli/push/channels/remove-where) | Remove push channel subscriptions matching filter criteria. |
