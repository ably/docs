Use the `ably bench` command group to run performance benchmarking tests. These commands enable you to measure publishing and subscribing throughput on Ably channels.

## Subcommands

| Command | Description |
| ------- | ----------- |
| [`ably bench publisher`](/docs/cli/bench/publisher) | Run a publisher benchmark test. |
| [`ably bench subscriber`](/docs/cli/bench/subscriber) | Run a subscriber benchmark test. |
