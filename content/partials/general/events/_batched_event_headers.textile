Batched events will have the following headers:

- x-ably-envelope-appid := the "app ID":https://support.ably.com/support/solutions/articles/3000063083 which the message came from
- content-type := the type of the payload. This will be @application/json@ or @application/x-msgpack@
- x-ably-version := the version of Reactor Event. At present this should be @1.2@
