<Code>
```shell
ably queues delete <QUEUENAME> [flags]
```
</Code>

Use the `ably queues delete` command to permanently remove an [Ably Queue](/docs/queues). This action cannot be undone and will delete all messages and configuration associated with the queue.

## Arguments

Pass the following arguments to the `ably queues delete` command:

### `<QUEUENAME>` <a id="queuename"/>

The name of the queue to delete, for example: `my-queue`.

## Flags

You can optionally pass the following flags to the `ably queues delete` command:

### `--app` <a id="app"/>

App ID or name to delete the queue from. If not specified, uses the current app from your CLI configuration.

### `--force | -f` <a id="force"/>

Force deletion without confirmation prompt. Use with caution as this action is irreversible.

### `--help | -h` <a id="help"/>

Show CLI help for this command.

## Global flags

The following global flags are supported using the `ably queues delete` command:

* [`--api-key`](/docs/cli#api-key)
* [`--access-token`](/docs/cli#access-token)
* [`--client-id`](/docs/cli#client-id)
* [`--endpoint`](/docs/cli#endpoint)
* [`--env`](/docs/cli#env)
* [`--host`](/docs/cli#host)
* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--token`](/docs/cli#token)
* [`--verbose`](/docs/cli#verbose)

## Examples

Delete a queue with confirmation:

<Code>
```shell
ably queues delete my-queue
```
</Code>

Delete a queue from a specific app:

<Code>
```shell
ably queues delete my-queue --app "My App"
```
</Code>

Delete a queue without confirmation:

<Code>
```shell
ably queues delete my-queue --force
```
</Code>
