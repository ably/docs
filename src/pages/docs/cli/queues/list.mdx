<Code>
```shell
ably queues list [flags]
```
</Code>

Use the `ably queues list` command to list all queues for an application.

## Flags

You can optionally pass the following flags to the `ably queues list` command:

### `--app` <a id="app"/>

The app ID to list queues for. If not specified, the currently selected app is used.

### `--limit` <a id="limit"/>

The maximum number of queues to return. The default value is `100`.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

List all queues:

<Code>
```shell
ably queues list
```
</Code>

List all queues in JSON format:

<Code>
```shell
ably queues list --json
```
</Code>
