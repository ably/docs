A small patch release addressing two issues reported after `v1.0.0`.

This release fixes a bug where sprinkles could be applied twice after a transient disconnect, and corrects a rounding error in the `meltWarning` timer that occasionally fired a few seconds early.

<Aside data-type='note'>
No action is required to upgrade. Existing cones pick up the fix automatically on reconnect.
</Aside>

For the full diff, see the [release notes](https://github.com/ably/ably-js/releases).
