<Code>
```shell
ably channels annotations publish <channel> <serial> <type> [flags]
```
</Code>

Use the `ably channels annotations publish` command to publish an annotation on a channel message.

## Arguments

Pass the following arguments to the `ably channels annotations publish` command:

### `<channel>` <a id="channel"/>

The name of the channel containing the message.

### `<serial>` <a id="serial"/>

The serial of the message to annotate.

### `<type>` <a id="type"/>

The annotation type. For example, `reactions:unique.v1` or `metrics:total.v1`.

## Flags

You can optionally pass the following flags to the `ably channels annotations publish` command:

### `--name | -n` <a id="name"/>

The annotation name. For example, an emoji name such as `thumbsup`.

### `--count` <a id="count"/>

The count value for the annotation. Used with `multiple.v1` type annotations.

### `--data` <a id="data"/>

A JSON payload to include with the annotation.

### `--encoding | -e` <a id="encoding"/>

The encoding of the annotation data.

### `--client-id` <a id="client-id"/>

A client ID to use when publishing the annotation.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Publish a simple annotation:

<Code>
```shell
ably channels annotations publish my-channel "01a2b3c4d5@1234567890000-0" "reactions:unique.v1"
```
</Code>

Publish a reaction with a name:

<Code>
```shell
ably channels annotations publish my-channel "01a2b3c4d5@1234567890000-0" "reactions:unique.v1" --name thumbsup
```
</Code>

Publish an annotation with a count:

<Code>
```shell
ably channels annotations publish my-channel "01a2b3c4d5@1234567890000-0" "metrics:multiple.v1" --name views --count 5
```
</Code>
