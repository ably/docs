Use the `ably channels delete` command to delete a message on an Ably Pub/Sub channel.

## Synopsis

<Code>
```shell
ably channels delete <channel-name> <message-serial> [flags]
```
</Code>

## Arguments

### `channel-name` <a id="channel-name"/> <RequiredBadge />

The name of the channel containing the message.

### `message-serial` <a id="message-serial"/> <RequiredBadge />

The serial of the message to delete.

## Flags

### `--description` <a id="description"/>

A description for the deletion.

### `--client-id` <a id="client-id"/>

A client ID to use when deleting the message.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — JSON output. Mutually exclusive flags.
* [`--verbose`](/docs/cli#verbose) — Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Delete a message:

<Code>
```shell
ably channels delete my-channel "01234567890:0"
```
</Code>

Delete a message with a description:

<Code>
```shell
ably channels delete my-channel "01234567890:0" --description "Removed inappropriate content"
```
</Code>

Delete a message and output the result in JSON format:

<Code>
```shell
ably channels delete my-channel "01234567890:0" --json
```
</Code>

Delete a message and output the result in formatted JSON:

<Code>
```shell
ably channels delete my-channel "01234567890:0" --pretty-json
```
</Code>
