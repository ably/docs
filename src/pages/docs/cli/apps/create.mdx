Use the `ably apps create` command to create a new Ably application in your account.

## Synopsis

<Code>
```shell
ably apps create --name <name> [flags]
```
</Code>

## Flags

### `--name` <a id="name"/> `(Required)`

The name of the application to create.

### `--tls-only` <a id="tls-only"/>

Restrict the application to TLS connections only. When enabled, all connections to this app must use TLS encryption.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — JSON output. Mutually exclusive flags.
* [`--verbose`](/docs/cli#verbose) — Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Create a new application:

<Code>
```shell
ably apps create --name "my-app"
```
</Code>

Create an application restricted to TLS connections only:

<Code>
```shell
ably apps create --name "secure-app" --tls-only
```
</Code>

Create an application and output the result in JSON format:

<Code>
```shell
ably apps create --name "My New App" --json
```
</Code>

Create an application using an access token environment variable:

<Code>
```shell
ABLY_ACCESS_TOKEN="YOUR_ACCESS_TOKEN" ably apps create --name "My New App"
```
</Code>
