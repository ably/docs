<Code>
```shell
ably rooms typing keystroke <room> [flags]
```
</Code>

Use the `ably rooms typing keystroke` command to send a typing indicator in an Ably Chat room.

## Arguments

Pass the following arguments to the `ably rooms typing keystroke` command:

### `<room>` <a id="room"/>

The name of the chat room to send the typing indicator in.

## Flags

You can optionally pass the following flags to the `ably rooms typing keystroke` command:

### `--auto-type` <a id="auto-type"/>

Keep the typing indicator active continuously. Defaults to `false`.

### `--duration | -D` <a id="duration"/>

The duration in seconds to keep the typing indicator active.

### `--client-id` <a id="client-id"/>

A client ID to use when sending the typing indicator.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Send a typing indicator:

<Code>
```shell
ably rooms typing keystroke my-room
```
</Code>

Keep the typing indicator active continuously:

<Code>
```shell
ably rooms typing keystroke my-room --auto-type
```
</Code>
