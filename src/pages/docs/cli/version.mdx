```shell
ably version
```
</Code>

Use the `ably version` command to display version information for the Ably CLI, including the CLI version, platform architecture, and Node.js version.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Display the CLI version:

<Code>
```shell
ably version
```
</Code>
