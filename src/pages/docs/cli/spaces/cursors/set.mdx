<Code>
```shell
ably spaces cursors set <space> [flags]
```
</Code>

Use the `ably spaces cursors set` command to set a cursor with position data in a space.

Use another terminal instance and the [`ably spaces cursors subscribe`](/docs/cli/spaces/cursors/subscribe) command to view cursor movements as they occur, or use [`ably spaces cursors get-all`](/docs/cli/spaces/cursors/get-all) to view all current cursor positions.

## Arguments

### `<space>` <a id="space"/>

The space name to set cursor in, for example: `example-space`.

## Flags

You can optionally pass the following flags to the `ably spaces cursors set` command:

### `--x` <a id="x"/>

The x coordinate for cursor position.

### `--y` <a id="y"/>

The y coordinate for cursor position.

### `--data` <a id="data"/>

The cursor data to set as a JSON string, for example: `{"name": "John", "color": "#ff0000"}`.

### `--simulate` <a id="simulate"/>

Simulate cursor movement every 250ms with random positions.

### `--duration | -D` <a id="duration"/>

Automatically exit after the given number of seconds. Use `0` to exit immediately after setting the cursor (default behavior).

## Global flags

The following global flags are supported using the `ably spaces cursors set` command:

* [`--token`](/docs/cli#token)
* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--api-key`](/docs/cli#api-key)
* [`--access-token`](/docs/cli#access-token)
* [`--client-id`](/docs/cli#client-id)
* [`--env`](/docs/cli#env)
* [`--host`](/docs/cli#host)
* [`--endpoint`](/docs/cli#endpoint)
* [`--verbose`](/docs/cli#verbose)

## Output

When setting a cursor, you will see confirmation that the cursor has been set with the specified data and position information.

## Examples

### Set a cursor with coordinates

Set a cursor at position x:100, y:200:

<Code>
```shell
ably spaces cursors set example-space --x 100 --y 200
```
</Code>

**Output:**
<Code>
```text
Using: App=example-app (Ab1CdE) • Key=example-key (Ab1CdE.FgHiJk)

Entered space: example-space
✓ Set cursor in space example-space with data: {"position":{"x":100,"y":200}}
Cursor set. Press Ctrl+C to exit.
```
</Code>

### Set a cursor with additional data

Set a cursor with position and additional metadata:

<Code>
```shell
ably spaces cursors set example-space --x 300 --y 400 --data '{"color":"red","name":"TestCursor"}'
```
</Code>

**Output:**
<Code>
```text
Using: App=example-app (Ab1CdE) • Key=example-key (Ab1CdE.FgHiJk)

Entered space: example-space
✓ Set cursor in space example-space with data: {"position":{"x":300,"y":400},"data":{"color":"red","name":"TestCursor"}}
Waiting 2s before exiting… Press Ctrl+C to exit sooner.
Duration elapsed – command finished cleanly.
```
</Code>

### Set a cursor with automatic exit

Set a cursor and exit after 5 seconds:

<Code>
```shell
ably spaces cursors set example-space --x 150 --y 250 --duration 5
```
</Code>

### Simulate cursor movement

Simulate random cursor movement:

<Code>
```shell
ably spaces cursors set example-space --simulate
```
</Code>
