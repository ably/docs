Use the `ably support` command group to get support and help from Ably. These commands provide access to AI assistance, contact information, and support resources.

## Subcommands

The following commands are available for support:

| Command | Description |
| ------- | ----------- |
| [`ably support ask`](/docs/cli/support/ask) | Ask a question to the Ably AI agent for help. |
| [`ably support contact`](/docs/cli/support/contact) | Contact Ably for assistance. |
| [`ably support info`](/docs/cli/support/info) | General support resources and documentation links. |
