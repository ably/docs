Use the `ably queues create` command to create a new queue for your Ably application.

## Synopsis

<Code>
```shell
ably queues create --name <name> [options]
```
</Code>

## Options

### `--name` <a id="name"/> <RequiredBadge />

The name of the queue to create.

### `--app` <a id="app"/>

The app ID to create the queue for. If not specified, the currently selected app is used.

### `--max-length` <a id="max-length"/>

The maximum number of messages the queue can hold. The default value is `10000`.

### `--region` <a id="region"/>

The region to create the queue in. The default value is `us-east-1-a`. See the [queues documentation](/docs/platform/integrations/queues) for more details.

### `--ttl` <a id="ttl"/>

The time-to-live for messages in the queue, in seconds. The default value is `60`. The maximum value is `3600`.

### `--json` <a id="json"/>

Output results as compact JSON. Mutually exclusive with `--pretty-json`.

### `--pretty-json` <a id="pretty-json"/>

Output results in formatted JSON. Mutually exclusive with `--json`.

### `--verbose | -v` <a id="verbose"/>

Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Create a queue:

<Code>
```shell
ably queues create --name "my-queue"
```
</Code>

Create a queue with custom settings:

<Code>
```shell
ably queues create --name "my-queue" --max-length 5000 --ttl 300
```
</Code>

Create a queue in a specific region:

<Code>
```shell
ably queues create --name "my-queue" --region eu-west-1-a
```
</Code>

Create a queue and output the result in JSON format:

<Code>
```shell
ably queues create --name "my-queue" --json
```
</Code>
