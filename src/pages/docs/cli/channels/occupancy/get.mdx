Use the `ably channels occupancy get` command to get the current occupancy for an Ably Pub/Sub channel.

## Synopsis

<Code>
```shell
ably channels occupancy get <channel> [flags]
```
</Code>

## Arguments

### `channel` <a id="channel"/>

The name of the channel to get occupancy for.

## Flags

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Get the current occupancy for a channel:

<Code>
```shell
ably channels occupancy get my-channel
```
</Code>

Get occupancy in JSON format:

<Code>
```shell
ably channels occupancy get my-channel --json
```
</Code>
