<Code>
```shell
ably spaces members get <name>
```
</Code>

Use the `ably spaces members get` command to get all members in a space.

## Arguments

Pass the following arguments to the `ably spaces members get` command:

### `<name>` <a id="name"/>

The name of the space to get members for.

## Flags

You can optionally pass the following flags to the `ably spaces members get` command:

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json)
* [`--pretty-json`](/docs/cli#pretty-json)
* [`--verbose`](/docs/cli#verbose)

## Examples

Get all members in a space:

<Code>
```shell
ably spaces members get my-space
```
</Code>

Get members in JSON format:

<Code>
```shell
ably spaces members get my-space --json
```
</Code>
