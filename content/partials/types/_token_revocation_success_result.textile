A @TokenRevocationSuccessResult@ contains information about the result of a successful token revocation request for a single target specifier.

h4.
  default: Properties

- <span lang="default">target</span> := The target specifier<br>__Type: @String@__

- <span lang="default">appliesAt</span> := The time at which the token revocation will take effect, as a Unix timestamp in milliseconds<br>__Type: @Number@__

- <span lang="default">issuedBefore</span> := A Unix timestamp in milliseconds. Only tokens issued earlier than this time will be revoked<br>__Type: @Number@__
