<Code>
```shell
ably push config clear-apns [flags]
```
</Code>

Use the `ably push config clear-apns` command to clear APNs push notification configuration for an app.

## Flags

You can optionally pass the following flags to the `ably push config clear-apns` command:

### `--app` <a id="app"/>

The app ID to clear configuration for.

### `--force | -f` <a id="force"/>

Skip confirmation prompt. Required when using `--json`. The default is `false`.

## Global flags

The following [global flags](/docs/cli#global-flags) are also available:

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Clear APNs configuration with confirmation:

<Code>
```shell
ably push config clear-apns
```
</Code>

Force clear APNs configuration:

<Code>
```shell
ably push config clear-apns --force
```
</Code>
