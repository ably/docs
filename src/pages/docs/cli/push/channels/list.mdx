Use the `ably push channels list` command to list push channel subscriptions.

## Synopsis

<Code>
```shell
ably push channels list --channel <channel> [flags]
```
</Code>

## Flags

### `--channel` <a id="channel"/> `(Required)`

The channel to list subscriptions for.

### `--device-id` <a id="device-id"/>

Filter subscriptions by device ID.

### `--client-id` <a id="client-id"/>

Filter subscriptions by client ID.

### `--limit` <a id="limit"/>

The maximum number of subscriptions to retrieve. Defaults to `100`.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

List all subscriptions for a channel:

<Code>
```shell
ably push channels list --channel my-channel
```
</Code>

Filter subscriptions by device:

<Code>
```shell
ably push channels list --channel my-channel --device-id "device123"
```
</Code>

List push channel subscriptions in JSON format:

<Code>
```shell
ably push channels list --channel my-channel --json
```
</Code>
