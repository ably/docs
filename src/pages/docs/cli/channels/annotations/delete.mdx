Use the `ably channels annotations delete` command to delete an annotation from a channel message.

## Synopsis

<Code>
```shell
ably channels annotations delete <channel-name> <message-serial> <annotation-type> [flags]
```
</Code>

## Arguments

### `channel-name` <a id="channel-name"/> <RequiredBadge />

The name of the channel containing the message.

### `message-serial` <a id="message-serial"/> <RequiredBadge />

The serial of the message to delete the annotation from.

### `annotation-type` <a id="annotation-type"/> <RequiredBadge />

The annotation type to delete.

## Flags

### `--name | -n` <a id="name"/>

The annotation name to delete. Use this to target a specific annotation when multiple annotations of the same type exist.

### `--client-id` <a id="client-id"/>

A client ID to use when deleting the annotation.

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — JSON output. Mutually exclusive flags.
* [`--verbose`](/docs/cli#verbose) — Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Delete a receipt annotation:

<Code>
```shell
ably channels annotations delete my-channel "01234567890:0" "receipts:unique.v1"
```
</Code>

Delete a reaction by name:

<Code>
```shell
ably channels annotations delete my-channel "01234567890:0" "reactions:unique.v1" --name thumbsup
```
</Code>

Delete a flag-type annotation such as a read receipt:

<Code>
```shell
ably channels annotations delete my-channel "01234567890:0" "receipts:flag.v1"
```
</Code>

Delete a distinct category annotation by name:

<Code>
```shell
ably channels annotations delete my-channel "01234567890:0" "categories:distinct.v1" --name important
```
</Code>

Delete an annotation and output the result in JSON format:

<Code>
```shell
ably channels annotations delete my-channel "01234567890:0" "receipts:flag.v1" --json
```
</Code>
