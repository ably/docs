Use the `ably spaces locations get` command to get all current locations in a space.

## Synopsis

<Code>
```shell
ably spaces locations get <name>
```
</Code>

## Arguments

### `name` <a id="name"/>

The name of the space to get locations for.

## Flags

## Global flags

* [`--json`](/docs/cli#json) or [`--pretty-json`](/docs/cli#pretty-json) — mutually exclusive output format flags.
* [`--verbose`](/docs/cli#verbose) — can be combined with either output format flag.

## Examples

Get all current locations in a space:

<Code>
```shell
ably spaces locations get my-space
```
</Code>

Get locations in JSON format:

<Code>
```shell
ably spaces locations get my-space --json
```
</Code>
