The Ably REST client libraries provide a convenient and simple API to interact directly with the "Ably REST API":/rest-api/. All official client libraries provide a consistent API across every language, offering "publish":/rest/channels#publish and "historical message retrieval":/rest/history, "basic and token authentication":/rest/authentication, "automatic handling of changing network conditions":https://knowledge.ably.com/routing-around-network-and-dns-issues, optional "symmetric encryption":/rest/encryption and access to "channel, presence and metrics history":/rest/history.
