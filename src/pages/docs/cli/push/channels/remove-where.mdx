Use the `ably push channels remove-where` command to remove push channel subscriptions matching filter criteria.

## Synopsis

<Code>
```shell
ably push channels remove-where --channel <channel> [options]
```
</Code>

## Options

### `--channel` <a id="channel"/> <RequiredBadge />

The channel to remove subscriptions from.

### `--device-id` <a id="device-id"/>

Filter subscriptions by device ID.

### `--client-id` <a id="client-id"/>

Filter subscriptions by client ID.

### `--force | -f` <a id="force"/>

Skip confirmation prompt. Required when using `--json`. The default is `false`.

### `--json` <a id="json"/>

Output results as compact JSON. Mutually exclusive with `--pretty-json`.

### `--pretty-json` <a id="pretty-json"/>

Output results in formatted JSON. Mutually exclusive with `--json`.

### `--verbose | -v` <a id="verbose"/>

Enable verbose logging. Can be combined with `--json` or `--pretty-json`.

## Examples

Remove all subscriptions for a device on a channel:

<Code>
```shell
ably push channels remove-where --channel my-channel --device-id "device123"
```
</Code>

Remove all subscriptions for a client on a channel:

<Code>
```shell
ably push channels remove-where --channel my-channel --client-id "user456"
```
</Code>

Remove push subscriptions and output in JSON format:

<Code>
```shell
ably push channels remove-where --channel my-channel --json
```
</Code>
